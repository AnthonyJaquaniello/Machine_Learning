# Machine Learning

## Jupyter notebook

Créer l'environnement conda : 

```bash
conda env create -f machine_learning_env.yml
```

Activer l'environnement : 

```bash
conda active ML
```

Lancer Jupyter lab :

```bash
jupyter lab
```
